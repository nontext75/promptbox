# Prompt Library MVP 기능정의서 v1.0

## 1. 서비스 개요

인터넷/SNS/커뮤니티 등에서 수집한 프롬프트를 개인 서재처럼 저장하고, 빠르게 검색하며, 수정/재활용할 수 있는 개인용 Prompt Database.

## 2. 핵심 목표

1. 프롬프트를 빠르게 저장한다.
2. 필요할 때 쉽게 찾는다.
3. 수정해서 다시 쓴다.
4. AI로 일부 내용을 변경해 새 버전을 만든다.

## 3. 핵심 사용자

- AI 활용 실무자
- 디자이너
- 마케터
- 개발자
- 개인 생산성 사용자

## 4. MVP 핵심 기능

### 4-1. 프롬프트 저장

입력:

- 텍스트 붙여넣기
- 제목 직접 입력(선택)

자동 처리:

- 제목 생성
- 카테고리 분류
- 한줄 설명 생성

저장 항목:

- title
- content
- category
- summary
- tags
- created_at
- updated_at

### 4-2. 검색 / 탐색

- 키워드 검색
- 카테고리 필터
- 최근순 정렬
- 즐겨찾기 보기

### 4-3. 상세 페이지

표시:

- 제목
- 설명
- 원본 프롬프트
- 태그
- 저장일

기능:

- 복사
- 수정
- 삭제
- 즐겨찾기

### 4-4. 편집 기능

프롬프트 내용 수정 가능.

### 4-5. AI 수정 기능

사용자 명령 예시:

- 이 부분만 짧게 바꿔줘
- 병원용으로 바꿔줘
- Midjourney용으로 바꿔줘
- 더 고급스럽게 바꿔줘

결과:

- 수정본 생성
- 새 버전 저장 가능

## 6. 화면 구성

### 페이지 1. 홈

- 검색창
- 최근 저장 목록
- 카테고리 메뉴
- 새 프롬프트 추가 버튼

### 페이지 2. 등록

- 텍스트 입력창
- 저장 버튼

### 페이지 3. 상세

- 프롬프트 보기
- 복사
- 수정
- AI 수정

### 페이지 4. 설정

- 테마
- 데이터 백업(선택)

## 7. 사용자 플로우

### 저장

프롬프트 붙여넣기 → 저장 → AI 자동 분류

### 사용

검색 → 상세 보기 → 복사

### 수정

상세 보기 → AI 수정 요청 → 새 버전 저장
